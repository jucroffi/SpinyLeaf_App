# SpinyLeaf_App

SpinyLeaf is a research driven design application that evaluates how early stage building geometry affects occupant wellbeing. The tool integrates Ladybug Tools, Honeybee, EnergyPlus, Radiance, and custom machine learning models to assess Comfort, Delight, and Social dimensions. It produces structured, evidence based feedback to support wellbeing oriented decision making in the early stages of architectural design.
